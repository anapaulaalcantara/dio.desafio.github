#CRIANDO UM ARQUIVO DE TEXTO LOCAL E ADICIONANDO AO GITHUB

CRIE UMA PASTA

NO GIT DASH USE O COMANDO PARA ADICIONAR O NOVO ARQUIVO NA VERSÃO LOCAL
	git add .
PARA ADICIONAR A DESCRIÇÃO DA ATUALIZAÇÃO
	git commit -m "Inclusão das anotações que descrevem as etapas desse desafio"
PARA EFETUAR O PUSH DAS ALTERAÇÕES LOCAIS PARA O REPOSITÓRIO ONLINE
	git push origin main
